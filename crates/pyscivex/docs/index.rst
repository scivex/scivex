pyscivex documentation
======================

**pyscivex** is the Python interface to the Scivex data-science toolkit.
One ``pip install pyscivex`` replaces numpy, pandas, scipy, scikit-learn,
matplotlib, and more — backed by a pure-Rust engine for maximum performance.
